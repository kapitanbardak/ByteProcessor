\doxysubsection{Detailed Description}
Interface for the subject in the observer pattern. 

The \doxylink{classISubject}{ISubject} class defines a standard interface for all subject objects, which are observed by observers. It includes methods to attach, detach, and notify observers, as well as to get the state of the subject.
