\doxysubsection{Detailed Description}
Class responsible for processing bytes from a source and writing results to a sink. 

The \doxylink{classByteProcessor}{Byte\+Processor} class is an implementation of an observer pattern, which listens to a source, processes the bytes received, and writes the processed data to a sink. The processing is done in a separate thread.
